# 能源企业重大新闻跟踪网站

这个目录是一版静态网站，可直接打开 `index.html` 查看。新闻数据集中维护在 `data/news-data.js`。

## 更新方式

1. 每月月初检查 23 家公司的官网新闻、公告、投资者关系页面。
2. 每家公司每季度至少保留 5 条重大事件，优先官网来源。
3. 官网当季不足 5 条时，补充新华社、国务院国资委、交易所公告、主流财经媒体等权威来源，并把 `sourceType` 标为 `权威媒体`。
4. 海外公司新闻统一改写为中文，不机械直译；保留日期、事项、影响和来源链接。
5. 只向 `events` 数组追加或修订事件，不需要改页面结构。

## 发布方式

如果只是本地打开，别人看不到。要让别人通过同一个网址访问，需要部署到 GitHub Pages、企业内网服务器、对象存储静态网站或其他托管服务。部署后，每月更新数据并重新发布，同一个网址会展示新内容。

## 推荐公网发布：GitHub Pages

1. 在 GitHub 新建一个公开仓库。
2. 把本目录内容推送到该仓库的 `main` 分支。
3. 在仓库 `Settings -> Pages` 中选择 `Deploy from a branch`。
4. Branch 选择 `main`，Folder 选择 `/root`，保存。
5. 公开网址通常是 `https://你的用户名.github.io/仓库名/`。
